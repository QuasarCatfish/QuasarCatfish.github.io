C++20 Big Number Library

Overview

This project aims to create a C++ Library for an efficient implementation of big numbers (larger than 64-bit). There are three categories of number classes that I want to create for this project: arbitrary-precision, fixed-precision, and fast-big (class names TBD). For the first two categories, both integer and floating-point classes will exist, while there will only exist floating-point numbers for fast-big. As a stretch goal, I would also like to create a complex number class (that hinges on any of these previous classes for numerical storage) to further flesh out this library.

Tasks and Features

Minimum Viable Product

  • Arbitrary-precision non-templated integer
    • Stores with absolute precision any integer number
    • All integral math operations can be performed on this number
    • Similar to Java's BigInteger
  • Arbitrary-precision non-templtaed floating-point
    • Stores with absolute precision (to a specified number of decimal places) any floating-point number
    • All real math operations can be performed on this number
    • Similar to Java's BigDecimal
  • Fixed-precision templated integer
    • Stores integer numbers up to a set number of bits (big_int<32> would mimic int)
  • Fixed-precision templated floating-point
    • Stores floating-point numbers up to a set number of bits (big_float<32> would mimic float)
  • "Big-fast" non-templated floating-point
    • Can store any floating-point number with variable precision
    • Aims to provide a fast but less precise version of the arbitrary-precision floating-point class
  • Integral and Real Math Function Libraries
    • A collection of math functions that can be applied to either integral or real numbers
    • Mimics standard library functions like std::sin and std::log

Stretch Goals

  • Wrapper class to facilitate Complex Numbers
  • Complex Math Function Library
  • Integration with C++'s random engine and distributions
